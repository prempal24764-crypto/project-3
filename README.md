## Housing Price Data Analysis (EDA)

### Description
This project performs Exploratory Data Analysis (EDA) on a housing dataset
to understand the factors affecting house prices.
Python libraries such as Pandas, NumPy, Matplotlib and Seaborn are used
for data analysis and visualization.

### Objectives
- Understand the housing dataset
- Perform data cleaning
- Visualize important features
- Analyze price trends

### Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- GitHub
